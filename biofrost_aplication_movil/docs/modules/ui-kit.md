# UI Kit

Propósito: Componentes reutilizables y tokens de diseño que garantizan consistencia visual.

Componentes base:

- `Button`, `Input`, `Card`, `Modal`, `Badge`, `Avatar`, `List`.

Tokens:

- Colores primarios/secondary (usar paleta corporativa descrita en docs), tipografía y espaciado.

Implementación:

- Mantener en `lib/src/ui` y exponer widgets temáticos (dark/light).

Lecturas recomendadas:

- Frontend UI: [IntegradorHub/frontend/src/components/ui](IntegradorHub/frontend/src/components/ui)

Enfoque móvil: Sí — componentes y tokens diseñados para interacción táctil y accesibilidad móvil.

Roles permitidos (UI): `Docente`, `Invitado` (los componentes no imponen permisos, pero el flujo los utiliza para mostrar/ocultar acciones).
